# when
### Syntax 
>Syntax of when in kotlin similar to case in java
 ``` ruby
 when(condition){
     key1 -> statement
     key2 -> statement
     else ->{statement}
 }
 ```
 # Loop
 >Syntax of ```For``` and ```while``` in ```kotlin``` similar to ```java```
 
 |  Type | Syntax  |
 |---|---|
